## 配置

- MacOS Mojave 10.14.3(Build 18D109)
- SMBIOS：Macmini8,1
- 主板：华擎Z370M-ITX/ac
- CPU：Intel Core i5-8400
- 核显：Intel UHD Graphics 630
- 独显：暂无
- 内存：海盗船复仇者 DDR4 3200 8G*2
- 固态硬盘：三星 SSD 960 EVO 250GB（Windows）
- 固态硬盘：三星 SSD 860 EVO 250GB（macOS）
- 电源：安钛克额定500W EAG500 PRO 模组电源
- 机箱：追风者 P300
- 鼠标🖱：罗技G102
- 键盘⌨️：ikbc C-104
- 显示器🖥：AOC 1080p
